# Fake-review-Detection
Detects fake and genuine reviews using NLP preprocessing, TF-IDF feature extraction, and machine learning classifiers. Evaluated with accuracy, precision, recall, and F1-score to support trustworthy online platforms.

## Features

* Text cleaning and preprocessing
* TF-IDF vectorization
* Classification using ML models
* Evaluation with Accuracy, Precision, Recall, F1-score
* Predicts authenticity of new reviews

## Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* NLP preprocessing

## Use Case

Helps e-commerce platforms detect spam reviews and improve trust in user feedback systems.

## How to Run

1. Upload dataset with `review` and `label` columns
2. Run notebook cells in order
3. Test predictions with custom review text
